Objetivos

Objetivo General: Con este Sistema de Votaciones daremos a conocer y facilitar a las personas al momento de votar sin ninguna confusión y no tenga ninguna dificultad al elegir a su candidato preferido.

Objetivos Específicos: Este sistema será creado para una mejor compresión para las personas analfabetas, para las que no poseen un buen conocimiento al momento de votar.

Con este sistema lograremos un mejor registro para no tener problemas del resultado para dar a conocer quien realmente gano la candidatura.
